## ✨ Features

### 📝 Text Management
- Add text with custom font, size, color, opacity, rotation
- 6 font families (Helvetica, Times Roman, Courier + Bold variants)
- Multi-line text support
- Apply to current page or all pages
- Click on PDF to set position

### 🔗 Link Management
- Add external URL hyperlinks with custom styling
- Add internal page-to-page navigation links
- **Remove ALL links** from PDF (hyperlinks + page links)
- **Remove specific links** by URL filter

### 💧 Watermark System
- **Add** text watermarks with full control (color, opacity, rotation, position, tiling)
- **Add** image watermarks (PNG/JPEG) with opacity and rotation
- **Remove** text watermarks by searching content
- **Remove** image watermarks by size filter or remove all

### 🖼️ Image Embedding
- Embed PNG/JPEG images into any page
- Control size, position, opacity, rotation
- Preview before adding

### 📄 Page Management
- Add blank pages (A4, Letter, Legal, A3, custom)
- Delete specific pages
- Rotate pages (90°, 180°, 270°)
- Merge multiple PDFs
- Insert pages at any position

### 🛠️ Advanced Tools
- Edit PDF metadata (title, author, subject)
- Set page background color
- Auto page numbering (multiple formats)
- PDF info display
- Modification history

### 📱 Mobile Friendly
- Responsive design for all screen sizes
- Touch-friendly controls
- Slide-out sidebar on mobile
